# 範例文件資料夾

請將您的範例文件放入對應的部門資料夾：

```
documents/
├── 人力資源部/
│   ├── 勞動基準法.pdf
│   ├── 員工請假辦法.docx
│   └── ...
├── 商品部/
│   ├── 商品規格標準.pdf
│   └── ...
├── 業務部/
│   ├── 業務開發手冊.pptx
│   └── ...
└── 教育訓練部/
    ├── 新人訓練手冊.pdf
    └── ...
```

## 支援的檔案格式
- PDF (.pdf)
- Word (.doc, .docx)
- PowerPoint (.ppt, .pptx)
- Excel (.xls, .xlsx)
- 文字檔 (.txt)

## 使用方式
將檔案放入對應資料夾後，系統會自動在知識庫頁面顯示。
